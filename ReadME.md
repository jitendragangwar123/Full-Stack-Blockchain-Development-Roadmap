```shell
For package.json - 
>>npm init
To install hardhat - 
>>npm install --save-dev hardhat
To run hardhat - 
>>npx hardhat
To install chai and others - 
>>npm install --save-dev @nomiclabs/hardhat-ethers ethers @nomiclabs/hardhat-waffle ethereum-waffle chai
To compile the smart contract -
>>npx hardhat compile
```
